# Auto_Java

## Opis projektu

Auto_Java to aplikacja napisana w języku Java, której celem jest prezentacja umiejętności programowania obiektowego oraz implementacji logiki w przejrzystej i skalowalnej strukturze projektu.

Projekt koncentruje się na modelowaniu obiektów związanych z domeną, zarządzaniu nimi oraz wykonywaniu operacji na danych w sposób zgodny z zasadami OOP.

---

## Główne funkcje

- Modelowanie obiektów 
- Operacje na danych (tworzenie, edycja, wyświetlanie)
- Separacja logiki biznesowej od warstwy prezentacji
- Czytelna i modularna struktura projektu

## Instalacja i uruchomienie

### 1. Klonowanie repozytorium

```
git clone https://github.com/MikolajZzz/Auto_Java.git
cd Auto_Java
```

### 2. Uruchomienie projektu

#### Opcja 1: IDE
Otwórz projekt w IntelliJ IDEA lub Eclipse i uruchom klasę główną.

#### Opcja 2: Maven
```
mvn clean install
mvn exec:java
```
